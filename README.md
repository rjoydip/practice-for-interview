# Practice for Interview

This repository contains **JavaScript** and **TypeScript** code `snippets`, `examples`, and `tricky` concepts often encountered in technical interviews.

## Project Structure

### Concepts

- **Generators**: `src/concepts/generator/`
- **Maps**: `src/concepts/map/`
- **Object Proxy**: `src/concepts/object/proxy.ts`
- **OOP**: `src/concepts/oop/`
- **Operators**: `src/concepts/exp_opr/`

### Examples

Practical coding problems and algorithms.

- **Anagram Check**: `src/examples/anagram/`

- **Array Flattening**: `src/examples/flat_array.ts`
- **Longest Substring**: `src/examples/length_of_longest_substring.ts`
- **String Manipulation**: `src/examples/word_reverse_&_cap.ts`

### Tricks

Demonstrations of JavaScript/Node.js runtime behaviors and quirks.

- **Tricks**: `src/tricks/`
